# us-baby-names-data-analysis
# Introduction to Data Analysis & Visualization with Python  
### Using the US Baby Names Dataset

This project is a beginner-friendly introduction to **data analysis** and **data visualization** using Python.  
It uses the classic **US Baby Names dataset**, which contains:

- Name  
- Year  
- Gender  
- Count (number of babies with that name)

Through this project, learners will explore how to work with real datasets using **Pandas**, **Seaborn**, and **Matplotlib**.

---

## What You Will Learn

By following the notebooks in this repository, you will learn how to:

###  Load and inspect datasets  
###  Clean and filter data  
###  Use conditions and selections in Pandas  
###  Group and aggregate data  
###  Create new datasets from existing ones  
###  Visualize trends using Matplotlib  
###  Perform mini-data-science projects
